Step 2: Create two Products in Stripe Dashboard > Products:

     - Yearly price: $99.00/year (recurring) → copy price_xxx ID

   Product: Enterprise
     - Monthly price: $29.99/month (recurring) → copy price_xxx ID
     - Yearly price: $250.00/year (recurring) → copy price_xxx ID

Step 3: Create Webhook Endpoint in Stripe Dashboard > Developers > Webhooks:
   - URL: https://spacenexus.com/api/stripe/webhooks
   - Events to listen for:
     * checkout.session.completed
     * customer.subscription.updated
     * customer.subscription.deleted
     * invoice.payment_succeeded
     * invoice.payment_failed
   - Copy the Signing Secret (whsec_xxx)

Step 4: Configure Customer Portal in Stripe Dashboard > Settings > Billing > Customer Portal:
   - Enable subscription cancellation
   - Enable plan switching
   - Enable payment method updates
   - Set default return URL: https://spacenexus.com/pricing

Step 5: Set environment variables in Railway:
   STRIPE_SECRET_KEY=sk_live_xxx
   STRIPE_WEBHOOK_SECRET=whsec_xxx
   STRIPE_PRICE_PRO_MONTHLY=price_xxx
   STRIPE_PRICE_PRO_YEARLY=price_xxx
   STRIPE_PRICE_ENTERPRISE_MONTHLY=price_xxx
   STRIPE_PRICE_ENTERPRISE_YEARLY=price_xxx
   NEXT_PUBLIC_APP_URL=https://spacenexus.com

Step 6: Test with Stripe CLI before going live:
   stripe listen --forward-to localhost:3000/api/stripe/webhooks
   stripe trigger checkout.session.completed

Step 7: Create promo codes in Stripe Dashboard > Products > Coupons:
   - PRODUCTHUNT: 50% off for 3 months (for Product Hunt launch)

2. CRON JOB FOR EMAIL NURTURE (Medium-High — Automates free-to-paid conversion)
---------------------------------------------------------------------------------
Reference: docs/EMAIL_NURTURE_SEQUENCE.md

Step 1: Set CRON_SECRET environment variable in Railway:
   CRON_SECRET=<generate a random 32+ character string>

Step 2: Set up a daily cron job (Railway cron, or external like cron-job.org):
   - Schedule: Once daily, e.g. 9:00 AM ET
   - Method: POST
   - URL: https://spacenexus.com/api/nurture/process
   - Header: Authorization: Bearer <your CRON_SECRET value>

   Example curl:
   curl -X POST https://spacenexus.com/api/nurture/process \
     -H "Authorization: Bearer YOUR_CRON_SECRET"

The endpoint processes all 7 nurture emails automatically:
   Day 1: Welcome + quick start guide
   Day 3: Premium feature highlight
   Day 5: Social proof (industry stats)
   Day 7: AI Insights spotlight
   Day 10: Case study
   Day 14: 14-day free trial offer
   Day 21: 20% discount with code SPACE20
