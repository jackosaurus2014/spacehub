## 4. DEFENSE PROCUREMENT DATA

### SAM.gov Opportunities API
- **API Documentation**: https://open.gsa.gov/api/get-opportunities-public-api/
- **Contract Opportunities**: https://sam.gov/opportunities
- **Endpoint**: `https://api.sam.gov/prod/opportunities/v2/search`
  - Format: JSON
  - Updates: Daily (active), Weekly (archived)
  - Free: Yes (10 requests/day public, 1000/day registered)
  - Authentication: API key required (free registration)
  - Parameters: `limit`, `api_key`, `postedFrom`, `postedTo`, `ptype`, `deptname`
  - Content: Federal contract opportunities, RFPs, solicitations
  - Filter: Use keywords "space", "satellite", "launch" for space defense
  - Source: [GSA Open Technology](https://open.gsa.gov/api/get-opportunities-public-api/)

### SAM.gov Contract Awards API
- **API Documentation**: https://open.gsa.gov/api/contract-awards/
- **Contract Data**: https://sam.gov/contract-data
  - Format: JSON
  - Updates: Real-time
  - Free: Yes
  - Authentication: SAM.gov account + API key required
  - Content: Award and IDV contract data
  - DoD Data: 90-day delay on revealed contracts
  - Filters: Sections, AND (&), OR (~), NOT (!) conditions, free text search
  - Response: 10 records/page default, 100 max with `limit` parameter
  - Source: [GSA Contract Awards API](https://open.gsa.gov/api/contract-awards/)

### FPDS.gov (Federal Procurement Data System)
- **Main Site**: https://www.fpds.gov/
- **API Specs**: https://www.fpds.gov/wiki/index.php/V1.3_FPDS-NG_Web_Service_Integration_Specifications
- **Web Services Doc**: https://www.fpds.gov/downloads/FPDS-Specifications-WebServices_Integration_Specifications_V1.5.pdf
  - Format: SOAP/XML
  - Updates: Real-time
  - Free: Yes
  - Authentication: User ID/Password required
  - Protocol: SOAP 1.1, WSDL, HTTP GET/POST
  - Content: Authoritative federal procurement data
  - Schemas: Award.xsd, IDV.xsd at https://www.fpds.gov/FPDS/schema/
  - Source: [FPDS](https://www.fpds.gov/)

### USAspending.gov API
- **API Docs**: https://api.usaspending.gov/docs/endpoints
- **Endpoint**: `https://api.usaspending.gov/api/v2/search/spending_by_category/`
- **Training**: https://www.usaspending.gov/data/Basic-API-Training.pdf
  - Format: JSON
  - Updates: Daily
  - Free: Yes (no authentication required)
  - Filters:
    - NAICS codes (336411 = Aircraft Manufacturing, 336414 = Guided Missiles, 517410 = Satellite Telecom)
    - Agency codes (DoD, Space Force, Air Force)
    - Date ranges
    - Award amounts
  - Endpoints:
    - `/api/v2/search/spending_by_award/` - Award details
    - `/api/v2/search/spending_by_category/naics/` - By industry
    - `/api/v2/transactions/` - Transaction-level data
  - GitHub: https://github.com/fedspendingtransparency/usaspending-api
  - Source: [USAspending API](https://api.usaspending.gov/)

### SBIR/STTR Programs
- **AFWERX SBIR/STTR**: https://www.afsbirsttr.af.mil/
- **Awards Database**: https://www.afsbirsttr.us/USAFSBIRSTTRAwards/
- **SBIR.gov Topics**: https://www.sbir.gov/topics/

## 6. DATA FIELDS & CATEGORIZATION

### Contract Data Fields (SAM.gov/FPDS/USAspending)
- **Award Information**: Contract number, PIID, award amount, obligation amount
- **Vendor Details**: Company name, DUNS, CAGE code, location
- **Agency Data**: Awarding agency, funding agency, office
- **Classification**: NAICS code, PSC (Product Service Code)
- **Dates**: Award date, start date, completion date
- **Description**: Award description, keywords
- **Place of Performance**: City, state, country
- **Contract Type**: FFP, CPFF, T&M, IDIQ

### News Article Fields
- **Metadata**: Title, publication date, author, source
- **Content**: Summary, full text, keywords
- **Categories**: Program type, agency, country
- **Entities**: Companies mentioned, officials quoted
- **Topics**: Contracts, launches, policy, exercises

### NAICS Codes for Space Defense
- **336411**: Aircraft Manufacturing
- **336414**: Guided Missile and Space Vehicle Manufacturing
- **336415**: Guided Missile and Space Vehicle Propulsion Unit Manufacturing
- **336419**: Other Guided Missile and Space Vehicle Parts Manufacturing
- **517410**: Satellite Telecommunications
- **541712**: R&D in Physical, Engineering Sciences (includes aerospace R&D)

### PSC Codes for Space (sample)
- **1510**: Aircraft, Fixed Wing
- **1610**: Missiles
- **1680**: Miscellaneous Aircraft Accessories
- **5821**: Radio/TV Communications Equipment
- **7220**: Satellites, Satellite Systems, Meteorological

## 9. TECHNICAL INTEGRATION NOTES

### RSS Feed Processing
```typescript
// Example fetcher structure
interface RSSSource {
  name: string;
  url: string;
  category: 'us-military' | 'international' | 'news' | 'procurement';
  updateInterval: number; // minutes
  parser: 'rss' | 'atom';
}

const sources: RSSSource[] = [
  {
    name: 'US Space Force',
    url: 'https://www.spaceforce.mil/RSS/',
    category: 'us-military',
    updateInterval: 30,
    parser: 'rss'
  },
  // ... more sources
];
```

### SAM.gov API Example
```bash
# Get space-related opportunities
curl "https://api.sam.gov/prod/opportunities/v2/search?api_key=YOUR_KEY&limit=100&q=space+satellite" \
  -H "Accept: application/json"
```

### USAspending API Example
```bash
# Get Space Force contracts by NAICS
curl -X POST "https://api.usaspending.gov/api/v2/search/spending_by_category/naics/" \
  -H "Content-Type: application/json" \
  -d '{
    "filters": {
      "agencies": [{"type": "awarding", "tier": "subtier", "name": "Space Force"}],
      "naics": ["336414"],
      "time_period": [{"start_date": "2024-01-01", "end_date": "2025-12-31"}]
    }
  }'
```

### Error Handling
- **Circuit breaker** for failed API calls (use existing `src/lib/circuit-breaker.ts`)
- **Fallback to cache** when sources unavailable (use `src/lib/api-cache.ts`)
- **Structured logging** for all fetches (use `src/lib/logger.ts`)
- **Rate limit tracking** for SAM.gov (10/day public tier)

## 10. DATA QUALITY & VALIDATION

### Source Reliability Tiers
- **Tier 1 (Official Government)**: USSF, DoD, SAM.gov, FPDS, USAspending
- **Tier 2 (Established Trade Press)**: SpaceNews, Defense News, C4ISRNET
- **Tier 3 (Analysis/Commentary)**: Breaking Defense, Defense One
- **Tier 4 (International)**: NATO, UK MOD, foreign defense ministries

### Validation Rules
- **Contracts**: Cross-reference SAM.gov with USAspending for accuracy
- **News**: Require Tier 1/2 source for "confirmed" status
- **Dates**: Validate future dates are reasonable (< 10 years out)
- **Amounts**: Flag contracts > $10B for manual review
- **Duplicates**: Hash title+date to detect duplicate articles
